# LEVTIA MAX Bot

MVP-бот для MAX, который:
- принимает лиды из MAX
- собирает имя, телефон и Telegram username
- показывает номер студии для сохранения
- отправляет заявку на email
- сохраняет заявки локально в JSON

## Запуск

1. Скопировать `.env.example` в `.env`
2. Заполнить переменные окружения
3. Установить зависимости:
   `npm install`
4. Запустить локально:
   `npm run dev`
5. Для production:
   `npm start`

## Webhook

После запуска и публикации по HTTPS на 443 порту нужно зарегистрировать webhook в MAX на URL:

`https://your-domain.com/webhook/max`

Подписка должна включать типы:
- `bot_started`
- `message_created`
- `message_callback`

## Поток клиента

1. Нажимает кнопку "Оставить контакты"
2. Вводит имя
3. Вводит телефон
4. Вводит Telegram username или жмёт "Пропустить"
5. Получает подтверждение и номер студии
